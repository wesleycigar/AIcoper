---

## 一、我们不卖什么

**AIcoper 不是「AI 便利店」。不是「团购省钱工具」。**

我们拒绝成为"拐杖制造商"。AI 是放大器，不是拐杖。

我们做的是**全球创造力基础设施**——让中度到重度的AI使用者，在一个地方找到最强的模型、最合适的操作系统、最真实的评测、和最值得信任的同行。

---

## 二、北极星

> **"扩展人类创造未来的能力与自由。"**

这是AIcoper从第一天就写下的使命，从未改变。

便宜是副产品——规模采购自然便宜。但使命不是便宜。使命是让每一个科研人员的基金申请多一次挑战、让每一个创业者的BP多一层锤炼、让每一个企业家的决策多看一个视角。

---

## 三、产品矩阵 V3.0：比较大脑 + 比较操作系统

### 🧠 比较大脑（模型评测）
覆盖全球主要模型：DeepSeek / 通义千问 / 豆包 / Kimi / 智谱GLM / 元宝混元 / Claude / ChatGPT

评测维度：
- 逻辑推理能力
- 专业水平（学术/编程/金融/法律）
- 拟人水平
- 情感能力
- 社会化认知
- 成本

### 🖥️ 比较操作系统（AI工作系统评测）⭐ 新增
覆盖主流AI编程/工作系统：Claude Code / Codex / CodeWhale / WorkBuddy / Kimi Agent / Cursor / Windsurf

评测维度：
- 自动化水平
- 逻辑推理深度
- 成本效率
- 量化交易水平
- 人类和谐度认知
- 专业适配度

**模式**：AIcoper提供初始专业评测 → 注册用户打分+评价 → 汇总形成"AI工具大众点评"

## 五、推广策略

1. **GitHub开源** — 代码公开，社区共建，创造OpenClaw式效应
2. **学术论文** — 快速发表于AI领域高影响力开放平台（arXiv / HuggingFace / OpenReview）
3. **网页版工具** — 真正链接到全球大模型和AI操作系统的统一入口

## 六、合规策略（全球化版）

- **国内层**：国产合规模型聚合（已有备案路径）
- **国际层**：Claude / ChatGPT 免费模型 ——标注为"全球模型参考"，仅提供评测和链接，不代理、不转售
- **用户自行使用**：网页版提供链接跳转，用户在自己的合规环境中使用
